# Batyr Watch Warranty MVP

Мини-сайт для проверки оригинальности товара и срока гарантии по серийному номеру.

## Что уже есть

- Страница для покупателя: `/warranty/`
- Проверка серийного номера
- Статусы: гарантия активна, истекла, не активирована, возврат, замена, заблокирован
- Админ-панель: `/admin/`
- База товаров/серийников
- Маскировка данных покупателя на публичной странице
- WhatsApp-кнопка для обращения по гарантии
- Демо-серийники через команду `seed_demo`

## Быстрый запуск на Windows

Откройте папку проекта в VS Code, затем в терминале выполните:

```bash
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py seed_demo
python manage.py runserver
```

После запуска откройте:

```text
http://127.0.0.1:8000/warranty/
```

Админ-панель:

```text
http://127.0.0.1:8000/admin/
```

## Демо-серийники

```text
BW-SUN-2026-000125
BW-TOM-2026-000088
BW-SUN-2024-000001
```

## Как добавить свой товар

1. Зайдите в `/admin/`
2. Откройте раздел **Товары / гарантии**
3. Нажмите **Добавить**
4. Введите серийный номер, модель, дату продажи, срок гарантии и данные покупателя
5. Сохраните

## QR-ссылка

Можно печатать QR-код, который ведёт на такую ссылку:

```text
https://batyrwatch.kz/warranty/?serial=BW-SUN-2026-000125
```

Тогда поле серийного номера заполнится автоматически.

## Что поменять перед реальным запуском

В `config/settings.py`:

- `SECRET_KEY` заменить на безопасный ключ
- `DEBUG = False`
- добавить домен в `ALLOWED_HOSTS`, например `batyrwatch.kz`
- для продакшена лучше перейти с SQLite на PostgreSQL

Также в `warranty/views.py` поменяйте номер WhatsApp:

```python
'whatsapp_number': '77000000000'
```

на ваш реальный номер.
